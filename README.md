# Projet InfoLine – DevOps

Ce projet illustre la mise en place d’une infrastructure cloud sur GCP
et d’une chaîne CI/CD dans le cadre du TP Administrateur Système DevOps.

Technologies :
- Google Cloud Platform
- Terraform
- Kubernetes (GKE)
- Cloud SQL PostgreSQL
- Cloud Storage
- Cloud Functions
- GitHub Actions
